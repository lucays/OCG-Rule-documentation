『包含○○』
=============

以下情况不能对应：

-  | 『玩家必须』进行的那些处理是对玩家适用的效果，不能对应。
   | 例：
   | 「`救祓少女圣母悼歌`_」的③效果发动时，可以连锁发动「`屋敷童`_」的①效果。「`八云断巳剑`_」或「`北极天熊五倍线充能`_」的②效果发动时，，不能连锁发动「`屋敷童`_」的①效果。

『把卡片破坏』『给与伤害』等
=============================

根据以下等信息可以在发动时即确定是否是『把卡片破坏』『给与伤害』的效果。

-  | 发动时根据场所、回合信息、自身信息确定效果处理的效果。
   | 例：
   | 魔法·陷阱卡的发动时，连锁发动『发动无效并破坏』的效果的场合，由于魔法·陷阱卡的发动无效并破坏的状况不当作在场上被破坏，不能连锁发动「`星尘龙`_」的①效果。已经在场上表侧表示存在的魔法·陷阱卡的效果发动时，连锁发动『发动无效并破坏』的效果的场合，是破坏场上的卡片，可以连锁发动「`星尘龙`_」的①效果。\ `25/8/23 <https://www.db.yugioh-card.com/yugiohdb/faq_search.action?ope=5&fid=11290&keyword=&tag=-1&request_locale=ja>`__

-  | 发动时根据cost确定效果处理的效果。
   | 例：
   | 「`No.90 银河眼光子卿`_」的②效果发动时，取除的X素材不是「`银河`_」卡的场合，就只能无效，不能破坏，不能连锁发动「`破坏干扰`_」。是的场合，可以连锁发动「`破坏干扰`_」。\ `18/5/11 <https://www.db.yugioh-card.com/yugiohdb/faq_search.action?ope=5&fid=21890&request_locale=ja>`__

   | 例：

-  | 必发效果强制发动时，可以确定是否是『把卡片破坏』『给与伤害』的效果。
